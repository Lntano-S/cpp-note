# 【项目目标】
设计并实现一个 BusCard 类，模拟公交卡的基本功能：充值、消费、查询余额，并统计所有卡的总余额以及累计充值总额。通过本项目，练习类的封装、构造函数、析构函数、静态成员等知识点。

# 【类的数据成员】

- 卡号（唯一，不可修改）
- 持卡人姓名（string类）
- 卡内余额
- 所有卡的累计充值总额（静态成员，每次充值累加）
- 所有卡当前余额的总和（静态成员，随充值、消费、销毁动态变化）

# 【类的成员函数】

- 构造函数
- 拷贝构造函数
- 析构函数
- 充值函数
- 乘车消费函数
- 查询余额函数
- 查询卡号函数
- 查询持卡人姓名函数
- 静态成员函数：获取累计充值总额
- 静态成员函数：获取所有卡当前总余额

# 【测试要求】

## 编写一个主函数，演示以下操作序列，并在每一步后输出关键信息（包括所有卡总余额）：

1. 创建两张公交卡，卡号分别为 1001（持卡人 Tom）和 1002（持卡人Jack）。
2. 输出当前所有卡总余额（应为 0）。
3. Tom 充值 100 元，Jack充值 50 元。
4. 输出每次充值后的总余额（应为 100 和 150）。
5. Tom消费 2.5 元，输出成功或失败、Tom余额、总余额（应为 97.5 和 147.5）。
6. Tom尝试消费 200 元（应失败，提示余额不足），输出总余额（不变，仍为 147.5）。
7. Jack 消费 5 元，输出Jack 余额、总余额（Jack 余额 45，总余额 142.5）。
8. 输出累计充值总额（应为 150 元）。
